# Cat Image Uploader

Всем любителям котиков привет! Создал свой мини проект для генерации изображений котиков с текстом и автоматической загрузки на Яндекс.Диск.

## Описание

Скрипт позволяет:
1. Сгенерировать изображение котика с любым текстом через сервис [Cat as a Service](https://cataas.com/)
2. Автоматически создать папку на Яндекс.Диске
3. Загрузить туда полученное изображение
4. Сохранить JSON-файл с данными нашего изображения

## Быстрый старт

### Предварительные требования

1. Python 3.6 или выше
2. Токен доступа к Яндекс.Диску


## Получение токена Яндекс.Диска

1. Перейдите на [страницу OAuth-приложений Яндекс](https://yandex.ru/dev/disk/poligon/)

2. Авторизируйтесь

3. Получите OAuth-токен

4. Сохраните токен - он понадобится для работы скрипта

### Следуйте инструкциям:

1. Введите текст, который должен отображаться на изображении котика

2. Укажите название папки на Яндекс.Диске (будет создана, если не существует)

3. Введите ваш OAuth-токен Яндекс.Диска



## Пример использования

```
Загрузка картинок котиков на Яндекс.Диск
========================================

Введите текст для котика: Hello Man
Введите название папки на Яндекс.Диске: Cat_Images
Введите токен Яндекс.Диска: ваш_токен_здесь

Обрабатываем текст: 'Hello Man'
Папка 'Cat_Images' готова
Файл загружен: Cat_Images/Hello_Man.jpg
Файл загружен: Cat_Images/Hello_Man_info.json

Готово!
Папка: Cat_Images/
Картинка: Cat_Images/Hello_Man.jpg
Информация: Cat_Images/Hello_Man_info.json

Программа завершена успешно
```

## Формат сохраняемых данных

Для каждого изображения создается JSON-файл с информацией:

```
{
  "text": "Hello Man",
  "filename": "Hello_Man.jpg",
  "size_bytes": 45678,
  "source": "cataas.com"
}
```


## Настройка

Скрипт использует следующие настройки по умолчанию:

- Формат изображения: JPG

- Максимальная длина имени файла: 50 символов

- Таймаут загрузки: 10 секунд

## Возможные проблемы и решения

### Ошибка: "Не удалось получить картинку"

- Проверьте подключение к интернету

- Убедитесь, что сервис [cataas.com](https://cataas.com/) доступен

- Попробуйте другой текст

### Ошибка: "Не удалось создать папку"

- Проверьте правильность [токена Яндекс.Диска]((https://yandex.ru/dev/disk/poligon/))

- Убедитесь, что у токена есть права на запись

- Попробуйте другое название папки

### Ошибка: "Токен не может быть пустым"

- Убедитесь, что вы получили OAuth-токен на странице Яндекс.OAuth
